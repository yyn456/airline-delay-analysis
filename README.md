# Airline Delay Analysis

## Dataset

The original dataset was omitted due to GitHub file size limits.

The dataset can be downloaded from Kaggle.

After downloading, place the CSV file inside the `data/` folder before running the notebook.

## Tools Used

* Python
* Google Colab
* Pandas
* YData Profiling
* Tableau
* GitHub

## Project Objectives

* Perform data profiling and quality assessment
* Clean and preprocess airline delay data
* Conduct exploratory data analysis (EDA)
* Generate business insights
* Build an interactive Tableau dashboard
